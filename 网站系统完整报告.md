## 四、清除缓存的方法

### 方法A：Cloudflare后台手动（5分钟）
1. 登录 https://dash.cloudflare.com
2. 进入 **Workers and Pages** → **claw-app-2026** 项目
3. 点击 **Caching** → **Configuration**
4. 点 **Purge Everything** 清除全部缓存

### 方法B：Cloudflare API（需要Token）
需要用户提供 Cloudflare Global API Key，命令：
```
curl -X DELETE "https://api.cloudflare.com/client/v4/zones/{zone_id}/cache/purge" \
  -H "Authorization: Bearer {TOKEN}" \
  -H "Content-Type: application/json" \
  -d '{"files":["https://claw-app-2026.pages.dev/assets/index-CP-Zg9Cl.css","https://claw-app-2026.pages.dev/assets/index-DmCeXBoo.js"]}'
```
