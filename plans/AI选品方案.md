# AI全网爆款选品系统 — 方案

## 核心思路
建一套爆款评估标准 → AI全网搜索 → 打分评估 → 出报告

## 爆款评估维度（AI评分模型）
每个维度0-100分，加权算出总爆款概率：

| 维度 | 权重 | 说明 |
|------|------|------|
| 需求热度 | 25% | 搜索量、话题量、讨论热度 |
| 竞争程度 | 20% | 同类商品数量、卖家数（越少越好） |
| 利润空间 | 20% | 国内批发价 vs 海外售价差 |
| 内容传播力 | 15% | 适合拍视频/直播展示的程度 |
| 趋势增速 | 10% | 近期关注度上升速度 |
| 复购潜力 | 10% | 是否消耗品、配件等 |

## 数据来源（全网搜，不需要API）
- TikTok / TikTok Shop — 搜品类关键词，看新视频互动量
- Shopee / Lazada / OZON — 新品数据
- Google Trends — 搜索趋势
- 1688（只看公开数据，不爬）— 批发价参考
- Amazon — 评论数量、评分

## 输出
选品报告卡片，每品显示：
- 商品名称 + 参考图
- **爆款概率：XX%**（醒目的大数字）
- 各维度评分雷达图
- 建议售价区间
- AI分析理由
- 目标平台建议

## 技术实现
1. **选品指令** → 你告诉我搜什么品类（童装/翡翠/制冷配件）
2. **AI搜索分析** → 搜各平台 → 提取商品特征
3. **评分模型** → 按维度打分 → 算爆款概率
4. **出报告** → 展示在Claw网站上
