# AI智能选品 — 数据来源方案

## 现状
- 大模型本身有训练数据，能生成具体产品
- 但知识有截止日期，最新爆款不知道
- 评分是"模拟值"（基于产品特征计算，非真实销量数据）

## 方案：AI刷数据做补充

### 方式1：派AI刷抖音/TK/1688
安排子AI（operator/overseas）执行浏览器自动化，定期刷数据：

**刷什么：**
- 抖音精选联盟 → 看高佣爆款
- TikTok Shop → 看销量榜/热卖榜
- 1688 → 搜品类看热销款和批发价
- OZON → 看俄罗斯市场爆款

**刷出来存哪里：**
- `backend/data/scraped_products.json` → 持久化存储
- 每天增量刷，不重复

**结构化存什么：**
```json
{
  "name": "具体产品名",
  "platform": "tiktok/1688/ozon",
  "priceCNY": 批发价,
  "priceOverseas": 海外售价,
  "sales": 销量（如有）,
  "category": "品类",
  "sourceUrl": "来源链接",
  "scrapedAt": "2026-05-14"
}
```

### 方式2：AI主动搜索（无API）
不调API，用浏览器自动化：
1. 打开 TikTok → 搜索品类关键词 → 截图/录屏
2. AI看图识别产品和价格
3. 存入数据库

### 方式3：人工补充
- 用户自己在选品页手动录入感兴趣的单品
- 录入的自动进入选品库

## 评分升级
有了真实数据后，评分可以：
- 需求趋势 → 对比 TikTok 搜索热度（AI观察）
- 竞争程度 → 看同类产品数量（AI数）
- 利润空间 → 1688价 vs 平台售价（真实对比）
