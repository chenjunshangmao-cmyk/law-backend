# Claw项目局域网共享指南

## 🎯 目的
将Claw外贸电商平台项目文件夹共享给局域网内的所有AI设备，方便团队协作和代码查看。

## 📁 项目结构
```
Claw/
├── frontend/          # React前端项目
├── backend/           # Node.js后端API
├── src/               # 源代码文件
├── data/              # 数据文件
├── deploy-package/    # 部署文件
├── share-server.js    # Node.js共享服务器
├── simple-share.py    # Python共享服务器
├── start-share.bat    # Node.js启动脚本
├── start-python-share.bat # Python启动脚本
└── 局域网共享说明.md   # 本文件
```

## 🚀 共享方案选择

### 方案1：Node.js共享服务器（推荐）
**特点**：
- 功能丰富，支持文件浏览、下载、API接口
- 美观的Web界面
- 需要Node.js环境

**启动方法**：
1. 双击运行 `start-share.bat`
2. 或手动执行：`node share-server.js`

### 方案2：Python共享服务器（简单）
**特点**：
- 无需安装额外依赖（使用Python内置HTTP服务器）
- 简单易用，适合快速共享
- 需要Python 3.x环境

**启动方法**：
1. 双击运行 `start-python-share.bat`
2. 或手动执行：`python simple-share.py`

## 🔧 故障排除

### 问题1：端口8080被占用
**解决方法**：
1. 修改脚本中的端口号（如改为8081）
2. 或终止占用8080端口的进程

### 问题2：其他设备无法访问
**检查步骤**：
1. 确认设备在同一局域网
2. 检查防火墙设置（允许8080端口）
3. 确认IP地址正确
4. 尝试关闭防火墙临时测试

### 问题3：Node.js/Python未安装
**解决方法**：
1. **Node.js**：访问 https://nodejs.org 下载安装
2. **Python**：访问 https://python.org 下载安装

## 📊 网络配置建议

### 防火墙设置（Windows）：
```powershell
# 允许8080端口入站
New-NetFirewallRule -DisplayName "Claw Share Port" -Direction Inbound -LocalPort 8080 -Protocol TCP -Action Allow
```

### 静态IP设置（推荐）：
为避免IP变化，建议为共享电脑设置静态IP：
1. 打开"网络和共享中心"
2. 点击"更改适配器设置"
3. 右键点击网络连接 → 属性
4. 选择"Internet协议版本4 (TCP/IPv4)" → 属性
5. 选择"使用下面的IP地址"
6. 设置固定IP（如：192.168.1.100）

## ⚠️ 安全注意事项

### 建议措施：
1. **仅限内网**：不要将服务器暴露到公网
2. **访问控制**：如需限制访问，可添加基础认证
3. **定期关闭**：使用完毕后关闭服务器
4. **敏感文件**：避免共享包含敏感信息的文件

### 添加基础认证（Node.js）：
```javascript
// 在share-server.js中添加
const basicAuth = require('basic-auth');

const auth = (req, res, next) => {
  const user = basicAuth(req);
  if (!user || user.name !== 'admin' || user.pass !== 'password') {
    res.set('WWW-Authenticate', 'Basic realm="Claw Share"');
    return res.status(401).send('需要认证');
  }
  next();
};

app.use(auth); // 在所有路由前使用
```
