## API端点

```
GET  /api/stream-proxy/plans          — 套餐列表
GET  /api/stream-proxy/regions        — 区域列表
GET  /api/stream-proxy/nodes          — 节点状态
POST /api/stream-proxy/order          — 创建订单
GET  /api/stream-proxy/orders         — 用户订单
GET  /api/stream-proxy/subscription   — 当前订阅
POST /api/stream-proxy/activate       — 激活订单
GET  /api/stream-proxy/config         — 代理配置
POST /api/stream-proxy/usage-log      — 使用记录
GET  /api/stream-proxy/stats          — 统计
GET  /api/stream-proxy/validate       — 订阅验证
POST /api/stream-proxy/nodes          — 添加节点(管理)
DELETE /api/stream-proxy/nodes/:id    — 移除节点(管理)
```
