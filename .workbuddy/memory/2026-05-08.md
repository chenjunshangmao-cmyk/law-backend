## 第二波修复：Render 部署持续失败 (2026-05-08 08:00)
- **现象**: 从22f864d(AI图像工具)开始，所有后端部署均失败
- **根因3(关键)**: `src/routes/aiTools.js` 包含8处TypeScript类型注解（`: Buffer[]`、`: Buffer`、`req.file!`），Node.js ESM直接报SyntaxError
- **根因4**: `multer` 包未加入 package.json dependencies，aiTools.js import multer 时报 MODULE_NOT_FOUND

- **修复**: 
  - aiTools.js: 移除全部 `: Buffer`/`: Buffer[]` 和 `!` 非空断言
  - package.json: 添加 `"multer": "^1.4.5-lts.1"`
  - package.json: 移除 postinstall 脚本（Chromium不应在构建时下载）
  - render.yaml: buildCommand 改为 `npm install`（去postinstall）
