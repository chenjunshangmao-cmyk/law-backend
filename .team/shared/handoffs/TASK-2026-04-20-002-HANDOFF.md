## ✅ 完成内容

### 1. TikTokPage AI 功能（新增区块）

**文件**: `frontend/src/pages/TikTokPage.tsx`

新增「AI 文案 & 图片生成」区域，位于系统状态卡片和登录区之间，包含：

#### 功能 A: AI 生成文案
- 调用 `POST /api/generate/text`
- 参数: `{ productName, productDescription, platform: 'tiktok', style }`
- 支持 4 种风格：专业简洁 / 轻松活泼 / 年轻潮流 / 高端奢华
- 返回 `{ title, description, features[], keywords[] }`
- **自动填入发布表单**（标题和描述）
- 结果展示区：标题 + 描述 + 特点列表 + 关键词标签

#### 功能 B: AI 生成图片描述
- 调用 `POST /api/generate/image`
- 参数: `{ productName, productDescription, style }`
- 返回 `{ description, imageUrl }`
- 结果展示区：描述文本 + 图片预览

### 2. API 类型更新

**文件**: `frontend/src/services/api.ts`

`api.generate.text` 和 `api.generate.image` 类型扩展为兼容模式（支持新旧两种调用参数）：
- 旧版 `SmartPublishPage` 使用 `{ prompt, type, language }` → 仍可用
- 新版 `TikTokPage` 使用 `{ productName, productDescription, platform, style }` → 新增

### 3. 依赖图标
- `Sparkles` 图标从 lucide-react 导入
