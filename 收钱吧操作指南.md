## 2. 激活流程

### 2.1 当前问题

**E13 错误原因**：激活码 `66172491` 对应的终端 `100111220054361978`（设备 `claw-web-new1`）已在其他商户下绑定，新设备 `claw-web-new2` 无法使用同一个码。

**解决方案**：收钱吧客户经理已提供新激活码 `81119079`（有效期至 2026-04-30），绑定到新设备 `claw-web-new2`。

### 2.2 激活步骤

**Step 1：确认代码已更新**

检查 `backend/src/config/shouqianba.js` 确认以下内容：

```javascript
defaultDeviceId: 'claw-web-new2',
storeDevices: {
  'claw-web-new2': {
    code: '81119079',  // ✅ 新激活码
    merchantId: null,
    storeSn: null
  }
}
```

**Step 2：确认 Render 已部署新代码**

访问：https://claw-backend-2026.onrender.com/api/shouqianba/status

返回应包含：
```json
{
  "deviceId": "claw-web-new2",
  "terminalSn": null,
  "status": "not_activated"
}
```

> ⚠️ 如果返回的还是 `"claw-web-new1"` 或 terminalSn 有值，说明 Render 还在用旧代码，需要手动在 Render Dashboard 点击 **"Save, rebuild, and deploy"**

**Step 3：调用激活接口**

浏览器访问（GET 请求，无需参数）：
```
https://claw-backend-2026.onrender.com/api/shouqianba/activate
```

**成功响应：**
```json
{
  "success": true,
  "terminalSn": "10011122005436XXXX",
  "terminalKey": "xxxxxxxxxxxxxxxx",
  "merchantId": "18956XXXXX",
  "storeSn": "000101010012002000XXXXXX",
  "deviceId": "claw-web-new2"
}
```

**失败响应（E13）：**
```json
{
  "success": false,
  "error": "该设备已绑定其他商户",
  "code": "E13"
}
```

→ 说明激活码和新设备ID不匹配，需联系客户经理确认。

## 4. 常见错误码

| 错误码 | 含义 | 原因 | 解决方案 |
|--------|------|------|----------|
| **E13** | 设备已绑定其他商户 | 同一终端号在不同商户下激活 | 更换新设备ID + 新激活码 |
| **E100** | 签名错误 | vendor_key 或签名算法错误 | 检查 vendor_key 配置和 MD5 签名逻辑 |
| **E101** | 参数错误 | 必填参数缺失或格式错误 | 检查请求参数 |
| **E102** | 商户不存在 | vendor_sn 有误 | 确认 vendor_sn = 91803325 |
| **E103** | 终端不存在 | terminal_sn 未激活 | 先完成激活流程 |
| **E104** | 订单号重复 | client_sn 重复 | 使用唯一订单号 |
| **E200** | 交易超时 | 网络问题 | 重试 |
| **E201** | 余额不足 | 终端账户余额不足 | 充值收钱吧账户 |

### 4.1 E13 彻底解决方案

1. **联系客户经理**（方健平：13817783799），说明：
   > "设备 `claw-web-new2` 需要激活，请提供一个新激活码"
2. **提供新设备ID**：`claw-web-new2`（不要用旧的 `claw-web-new1`）
3. 获取新激活码后，按「激活流程」重新激活
4. 激活成功后更新代码中的 `testCode` 和 `storeDevices`

## 5. Render 环境变量配置

### 5.1 必需的环境变量

| 变量名 | 说明 | 值示例 |
|--------|------|--------|
| `DATABASE_URL` | PostgreSQL 连接字符串 | `postgresql://user:pass@host:5432/claw` |
| `JWT_SECRET` | JWT 签名密钥 | 随机 32+ 字符字符串 |
| `SHOUQIANBA_SEED_TERMINAL` | 终端种子信息（base64） | 见下方 |
| `ALLOWED_ORIGINS` | 允许的前端域名 | 多个用逗号分隔 |

### 5.2 SHOUQIANBA_SEED_TERMINAL 格式

```json
{
  "claw-web-new2": {
    "terminalSn": "10011122005436XXXX",
    "terminalKey": "xxxxxxxxxxxxxxxx",
    "merchantId": "18956XXXXX",
    "storeSn": "000101010012002000XXXXXX",
    "deviceId": "claw-web-new2"
  }
}
```

→ base64 编码后填入 Render 环境变量

### 5.3 查看当前 Render 终端状态

```
https://claw-backend-2026.onrender.com/api/shouqianba/status
```
