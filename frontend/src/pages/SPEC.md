# 社媒发布中心 - 设计规格

## 1. 概念与愿景

统一掌控三个平台的发布工作台。界面像一个"多合一指挥台"，左侧选平台切账号，中间选发布模式，右侧填内容发出去。清晰、高效、无缝切换。

## 2. 设计语言

- **美学方向**: 深色科技风格 + 平台品牌色点缀，干净专业
- **色彩**:
  - TikTok: `#FF0050` / `#00F2EA`
  - YouTube: `#FF0000`
  - 背景: `#0F1117`
  - 卡片: `#1A1D27`
  - 边框: `#2A2D3A`
  - 主文字: `#F1F3F5`
  - 次文字: `#8B90A0`
  - 成功: `#22C55E`
  - 警告: `#F59E0B`
  - 错误: `#EF4444`
- **字体**: Inter / -apple-system（无衬线）
- **布局**: 三段式（顶部平台切换 → 中部模式+账号+内容 → 底部预览/历史）

## 3. 页面结构


## 4. 三个平台 × 四种模式

### 平台
| 平台 | 图标 | 品牌色 | 核心功能 |
|------|------|--------|---------|
| TikTok Shop | 🎵 | #FF0050 | 商品上传、定价、库存 |
| TikTok Web | 🕶️ | #FF0050 | 视频发布、文案生成 |
| YouTube | ▶️ | #FF0000 | 视频上传、Google OAuth授权 |

### 发布模式
| 模式 | 适用平台 | 说明 |
|------|---------|------|
| 手动发布 | 全平台 | 填表单 → 浏览器自动化执行 |
| 半自动发布 | TK Shop / TK Web | AI生成内容 → 人工确认 → 发布 |
| 全自动发布 | TK Shop / TK Web | 选品→AI→定价→发布 全链路自动 |
| Google Cloud OAuth | YouTube | 独立授权管理（绑定YouTube账号） |

## 5. 账号管理

- 显示账号列表（平台、名称、登录状态、最后登录时间）
- 状态：✅ 已登录 / ⏳ 待验证 / ❌ 已失效
- 支持添加/删除/刷新状态
- 多账号切换（下拉选择当前发布账号）

## 6. 内容表单字段

**通用**：标题、描述、图片/视频、价格、成本

**TikTok Shop 专属**：库存、分类、标签、重量、运费

**TikTok Web 专属**：话题标签、视频时长、封面图

**YouTube 专属**：隐私设置（公开/不公开/私密）、缩略图、字幕

## 7. 技术实现

- 框架：React + TypeScript
- API：对接 `/api/browser/*`（TikTok/YouTube浏览器自动化）、`/api/publish/*`（发布流程）、`/api/accounts/*`（账号管理）
- 状态管理：React useState + useContext
- 无需额外依赖
